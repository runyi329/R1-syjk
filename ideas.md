## 1. 现代极简主义 (Modern Minimalist) - 选中方案

**设计运动**: 受瑞士国际主义风格 (Swiss Style) 和现代扁平化设计 (Flat Design 2.0) 启发。

**核心原则**:
- **清晰至上**: 数据是主角，去除所有不必要的装饰，让用户专注于数字和趋势。
- **网格系统**: 严格的网格布局，确保信息层次分明，易于阅读。
- **功能性色彩**: 颜色仅用于区分数据类别和强调关键信息（如推荐/不推荐），而非装饰。
- **留白艺术**: 大量使用留白来分隔内容块，减少认知负荷。

**色彩哲学**:
- **背景**: 纯白 (#FFFFFF) 或极浅灰 (#F8F9FA)，营造干净、专业的氛围。
- **主色**: 深蓝 (#2C3E50) 代表专业和信任。
- **辅助色**: 
  - 翡翠绿 (#27AE60): 代表低庄家优势（推荐）。
  - 琥珀黄 (#F39C12): 代表中等庄家优势（谨慎）。
  - 绯红 (#C0392B): 代表高庄家优势（危险/不推荐）。
- **文字**: 深灰 (#333333) 用于正文，浅灰 (#7F8C8D) 用于辅助说明。

**布局范式**:
- **非对称平衡**: 避免死板的完全居中，使用左侧导航/说明 + 右侧交互图表的布局。
- **卡片式设计**: 每个数据模块（如主要投注、对子、边注）封装在独立的卡片中，带有微妙的阴影。
- **响应式流体**: 布局随屏幕宽度平滑流动，确保在移动端和桌面端都有最佳体验。

**标志性元素**:
- **大号数字**: 关键数据（如1.06%）使用超大字号展示，作为视觉焦点。
- **细线条**: 使用细线条分隔内容，保持轻盈感。
- **圆形图表**: 使用环形图或半圆图来展示百分比，直观且美观。

**交互哲学**:
- **即时反馈**: 鼠标悬停在图表上时，立即显示详细数据和提示。
- **平滑过渡**: 切换数据视图时，使用淡入淡出或滑动动画，避免突兀的跳变。
- **探索式发现**: 初始只显示核心数据，用户点击或悬停可展开更多细节。

**动画**:
- **微交互**: 按钮点击、卡片悬停时的轻微缩放或阴影变化。
- **数据加载**: 图表加载时使用生长动画（如柱状图从0升起，环形图绘制过程）。
- **缓动曲线**: 所有动画使用 ease-out 曲线，感觉自然流畅。

**排版系统**:
- **标题**: Inter 或 Roboto，粗体，字间距微调，展现现代感。
- **正文**: Inter 或 Open Sans，易读性强。
- **数据**: JetBrains Mono 或 Roboto Mono，等宽字体确保数字对齐。

## 2. 赛博朋克霓虹 (Cyberpunk Neon)

**设计运动**: 赛博朋克 (Cyberpunk) 和复古未来主义 (Retro-Futurism)。

**核心原则**:
- **高对比度**: 深色背景配高亮霓虹色，模拟赌场夜景和电子屏幕。
- **科技感**: 使用故障艺术 (Glitch Art)、扫描线、发光效果。
- **沉浸式**: 营造一种置身于高科技赌场的氛围。

**色彩哲学**:
- **背景**: 深紫 (#1A0B2E) 或午夜蓝 (#0F172A)。
- **霓虹色**: 
  - 激光绿 (#00FF9D): 推荐。
  - 赛博黄 (#FFD700): 警告。
  - 霓虹粉 (#FF00FF): 危险。

**布局范式**:
- **HUD界面**: 模仿抬头显示器 (Heads-Up Display) 的布局，数据悬浮在背景之上。
- **模块化**: 强烈的边框和分割线，像控制面板。

**标志性元素**:
- **发光边框**: 元素周围带有辉光。
- **网格背景**: 背景中有淡淡的网格线或电路图纹理。

**交互哲学**:
- **机械感**: 按钮点击有清脆的视觉反馈。
- **高亮聚焦**: 鼠标悬停时，元素高亮发光。

**动画**:
- **故障效果**: 偶尔的画面抖动或色彩偏移。
- **扫描效果**: 数据加载时像扫描仪一样逐行显示。

**排版系统**:
- **标题**: Orbitron 或 Rajdhani，具有科技感的字体。
- **数据**: Share Tech Mono，像电子显示屏的字体。

## 3. 奢华装饰艺术 (Luxury Art Deco)

**设计运动**: 装饰艺术 (Art Deco) 和盖茨比风格 (Gatsby Style)。

**核心原则**:
- **优雅**: 强调线条的流畅和几何图形的对称。
- **奢华**: 使用金色、黑色、大理石纹理，体现高端赌场的质感。
- **精致**: 注重细节装饰，如边框花纹、字体衬线。

**色彩哲学**:
- **背景**: 丝绒黑 (#111111) 或深墨绿 (#013220)。
- **金属色**: 
  - 香槟金 (#D4AF37): 强调和高贵。
  - 银色 (#C0C0C0): 辅助信息。
  - 珍珠白 (#F5F5F5): 正文。

**布局范式**:
- **对称美学**: 强调中心对称，如同经典的建筑结构。
- **框架式**: 内容被精美的边框包围。

**标志性元素**:
- **几何纹样**: 扇形、菱形、放射状线条作为装饰。
- **衬线字体**: 优雅的衬线体，体现传统和经典。

**交互哲学**:
- **优雅从容**: 动画缓慢而流畅，如同丝绸飘动。
- **质感反馈**: 点击按钮有金属光泽流动的效果。

**动画**:
- **淡入淡出**: 页面元素优雅地显现。
- **线条生长**: 装饰线条慢慢延伸。

**排版系统**:
- **标题**: Playfair Display 或 Cinzel，高贵的衬线体。
- **正文**: Lato 或 Montserrat，保持现代易读性但带有优雅感。
